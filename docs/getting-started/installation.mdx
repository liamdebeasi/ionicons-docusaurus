# Installation

## Package Manager

The recommended way to install `ionicons` is to use a package manager.

<Tabs>
<TabItem value="npm" label="npm">

```shell
npm install ionicons
```
</TabItem>
<TabItem value="pnpm" label="pnpm">

```shell
pnpm install ionicons
```
</TabItem>
<TabItem value="yarn" label="Yarn">

```shell
yarn add ionicons
```
</TabItem>
</Tabs>

## CDN

Ionicons can also be loaded from a Content Delivery Network (CDN). Place the following `<script>` near the end of your page, right before the closing `</body>` tag.

```html
<script type="module" src="https://unpkg.com/ionicons@latest/dist/ionicons/ionicons.esm.js"></script>
```

:::note
The CDN links use the `latest` tag which always points to the latest version of Ionicons. We recommend pointing to a specific version, such as `6` instead of `latest`.
:::
